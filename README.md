# FiftyOne Enterprise Agriculture Workshop
## October 2025

This repo contains Jupyter notebooks you can use to explore the **Crops3D** and **Robusta Coffee Leaf (RoCoLe)** datasets.

## Crops3D
The `crops3d_fiftyone.ipynb` notebook will:
- Import the crops3d dataset from HuggingFace into FiftyOne
- Compute geometric embeddings
- Compute the FiftyOne embeddings visualization

## RoCoLe
The `rocole_fiftyone.ipynb` notebook will:
- Import the Robusta Coffee Leaf dataset from Huggingface into FiftyOne
- Compute embeddings and the embeddings visualization
- Train a lightweight PADIM model
- Compute model evaluation metrics
